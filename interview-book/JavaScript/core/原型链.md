# 原型链

```js

- Prototype 原型 ｜ 原型对象
1. Prototype 它是【函数】的一个属性
2. Prototype 是一个【对象】
3. 当我们创建函数的时候会默认添加 Prototype 这个属性


- __proto__ 原型链, 隐式原型
1. __proto__ 它是【对象】的一个属性
2. 指向构造函数的 Prototype
3. obj.__proto__ === Test.prototype

原型链的顶层 Object.prototype.__proto__ === null


function Test (){}
let obj = new Test()

obj : {
    a:1,
    __proto__: Test.prototype = {
        b:2
        __proto__: Object.prototype = {
            c:3
            __proto__: null
        }
    }
} // obj 可以依次找到 abc


```
